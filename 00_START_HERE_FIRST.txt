## 🔧 常见定制

### 改变内核加载地址
编辑 `Makefile`:
```makefile
LDFLAGS = -m elf_i386 -Ttext 0x2000  # 改为 0x2000
```

### 改变 VGA 颜色
编辑 `kernel/kernel.c`:
```c
#define VGA_COLOR_BLACK 0
#define VGA_COLOR_WHITE 7
// 改为你想要的颜色组合
```

### 增加 QEMU 内存
编辑 `Makefile`:
```makefile
run: $(OS_IMG)
	$(QEMU) -fda $< -m 256M  # 改为 256M
```

## 🆘 问题排查

### 编译失败
```bash
# 重新安装所有依赖
sudo apt-get update
sudo apt-get install build-essential nasm gcc-i686-linux-gnu binutils-i686-linux-gnu qemu-system-x86
```

### WSL 不可用
```powershell
# 在 PowerShell (管理员) 中重新安装
wsl --install
wsl --update
```

### QEMU 无法启动
```bash
# 重新编译
cd /mnt/c/Users/mfanq/OneDrive/Desktop/cos
make clean
make all
# 检查 build/os.img 是否存在
ls -lh build/os.img
```
